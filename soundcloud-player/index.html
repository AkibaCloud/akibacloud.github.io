# SoundCloud-Player

Install On Chrome (Edge, Opera, Brave) / Firefox (Experimental)

Links [ Source / Wiki Page / Donate / Google Form ] 





This is a showcase for the use of the Global shortcut feature.
You can control the SoundCloud audio by pressing keyboard shortcuts while playing the game.
However, depending on what you are using or playing, some software or games (such as major games like League of Legends) might block your shortcut inputs.
