# SoundCloud-Player

Install On Chrome (Edge, Opera, Brave) / Firefox (Experimental)

Links [ Source / Wiki Page / Donate / Google Form ] 





This is a showcase of SoundCloud Player Global shortcut feature.
You can control the soundcloud audio by pressing keyboard shortcuts, while you play the game.
However, depends on what you using/playing but some software/game (major game like league of legends) might block your shortcut inputs and I don't think can fix it.
